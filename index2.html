Hands-on Projects

Throughout the courses, students will work on hands-on projects to apply their knowledge and skills. These projects aim to provide practical experience in web development and cybersecurity.

  • Building a secure login system for a web application
  • Implementing cross-site scripting (XSS) and cross-site request forgery (CSRF) attacks
  • Performing penetration testing on web applications
  • Developing a secure RESTful API
